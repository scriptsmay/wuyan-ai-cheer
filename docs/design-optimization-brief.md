# 无言 AI 应援海报设计优化建议

> 基于当前工作空间项目 `wuyan-ai-cheer`（`src/lib/card-renderer.ts`）与现有设计系统（`src/styles/tokens.css`）给出的可落地建议。

## 1. 主文案字体层级修正

- **现状**：`card-renderer.ts` 中 `FONT_TITLE_MAIN` 实际使用 `HarmonyOS Sans SC`，但注释写的是“主标题：黄油体”。截图里的大标题也显得偏瘦、偏素，缺少“应援”情绪。
- **建议**：把主文案（如“日常打卡，今天也来给你加油啦！”）改为 `ZCOOL QingKe HuangYou`（黄油体），字号从 88px 提升到 96-104px，行高收紧到 1.05-1.1，让海报一眼就是“情绪句”。
- **实现点**：修改 `src/lib/card-renderer.ts` 第 28 行 `FONT_TITLE_MAIN` 的字体族为 `FAMILY_TITLE`。

## 2. 提升中间视觉密度

- **现状**：主文案与情绪标签之间有大片空白，整体视觉重心被“拉空”。
- **建议**：
  - 把主文案上移 60-80px，情绪标签与数据 refs 整体上移，压缩中间空白。
  - 在主文案背后加入 subtle “信号波纹”装饰（同心弧线或 3-4 条扩散线），呼应 “FAN SIGNAL STATION” 主题，也填补中宫。
  - 可在主文案下方加一条 1px 的品牌分隔线或一枚小型信号塔图标。

## 3. 背景网格弱化

- **现状**：青色斜线网格 `globalAlpha = 0.12` 太抢眼，和文字竞争注意力。
- **建议**：把网格透明度降到 0.04-0.06，或改用从左上角发散的微弱径向光晕打底。网格应该是“氛围”，不是“主角”。
- **实现点**：调整 `drawBackground` 中 `context.globalAlpha` 与 stroke 颜色。

## 4. 情绪标签视觉化

- **现状**：“☀️ 元气满满”只是橙色文字，在深色背景上识别度弱。
- **建议**：把它做成圆角 pill / badge，背景使用对应心情色（victory 金黄、low 紫、daily 青绿、hope 红），文字用深色；整体像一枚“情绪贴纸”。这样也与 web UI 中 `mood-victory`、`mood-daily` 等 color tokens 保持一致。
- **实现点**：在 `drawMainCopy` 中给 `emojiCaption` 加圆角背景块，颜色可基于 mood 参数映射。

## 5. 二维码与主体建立关联

- **现状**：右下角二维码孤立，像后期贴图。
- **建议**：
  - 在二维码上方加一行小字“扫码接收今日信号”。
  - 给二维码加一个细线框（1px cyan）或半透明背景块，让它成为海报视觉系统的一部分。
  - 二维码颜色可统一为 cyan/white，避免纯黑破坏画面。
- **实现点**：修改 `drawFooter` 中 QRCode 生成参数与绘制逻辑。

## 6. 顶部状态行降级

- **现状**：日期、连续天数、累计天数使用高饱和 `#00D4FF` + 32px，跳脱感太强，抢走了主文案的焦点。
- **建议**：把状态行颜色从 `#00D4FF` 降到 `#8DE7FF` 或 `#7AA8B8`，字号降到 26-28px，位置从标题下方移到更 subtle 的 badge 或右对齐。
- **实现点**：调整 `FONT_SUBTITLE_STATUS` 字号与 `buildCheckinRuns` 中的颜色。

## 7. 底部信息对比度提升

- **现状**：底部“生成日期”等使用 `#587483`，在 `#0A0E1A` 背景上对比度不足，保存到相册后更难辨认。
- **建议**：把底部文字颜色提升到 `#9FB4C0` 或 `#B0C4D0`，字号保持 22px 或略增到 24px，并继续用 `JetBrains Mono` + `Noto Sans SC` 混排。
- **实现点**：调整 `drawFooter` 中 `FONT_BODY_LABEL` 与 `FONT_MONO_META` 的颜色。

## 8. 按心情切换主题色（可选扩展）

- **现状**：海报永远是 cyan + orange，没有利用项目已有的 `victory`、`low`、`daily`、`hope` 心情色盘。
- **建议**：把 cyan/orange 作为 `daily` 默认，根据 `mood` 切换主强调色：
  - `daily` → 青 `#00D4FF` + 橙 `#FF9F4D`
  - `victory` → 金黄 `#F4C95D` + 深红/黑
  - `hope` → 珊瑚红 `#FF5A5F` + 白
  - `low` → 紫 `#7756A8` + 青
- **实现点**：需要在 `CardInput` 中传入 `mood`，并在 `card-renderer.ts` 建立 `MOOD_ACCENT` 颜色映射表。

---

## 优先级建议

| 优先级 | 改动 | 预计影响 |
|--------|------|----------|
| P0 | 黄油体主文案 + 背景网格弱化 | 情绪表达立刻提升，文字更突出 |
| P1 | 情绪标签 pill 化 + 二维码关联 | 信息层级更清晰，品牌感更强 |
| P2 | 状态行降级 + 底部对比度提升 | 阅读舒适度提升 |
| P3 | 心情主题色系统 | 长期体验差异化，需要接口/数据配合 |
